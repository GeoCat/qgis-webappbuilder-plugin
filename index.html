QGIS Web App Builder

A QGIS Plugin to create rich OL3-based web apps

Download .zip Download .tar.gz View on GitHub

A QGIS plugin to easily create web apps based on the OpenGeo Suite.

To install the most recent development version download this file and unzip it in your QGIS plugins folder (~/.qgis2/python/plugins/, you may need to create this folder)

Usage is documented here

Sample data and examples apps based on it are available. You can download the files with the app definitions used to build up those examples, and the corresponding layers data. Use them to explore the capabilities of the plugin and create your own apps.

We also have tutorials
